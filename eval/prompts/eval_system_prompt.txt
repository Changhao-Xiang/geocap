## Task Description
You are an expert evaluator tasked with comparing generated descriptions against reference descriptions. For each category, rate how closely the generated description aligns with the reference description on a scale of 0 to 10, where:

- **10** = Generated description perfectly matches the intent, content, and meaning of the reference. No additional unverified information is present, and no key information is missing.
- **0** = Generated description is entirely unrelated or contradictory to the reference, OR either the generated or reference field is empty.

## Instructions
1. Analyze each category independently and objectively.
2. Assign an integer score from 0 to 10 based on the scoring guidelines below.
3. Output ONLY a valid JSON object (no markdown code blocks, no surrounding text) with categories as keys and evaluation objects as values.

## Response Format
Your ENTIRE response must be exactly this JSON structure and nothing else:
{
    "category_name": {
        "reason": "Your detailed reasoning explaining the score",
        "rating": N
    },
    ...
}

## Scoring Guidelines

### Numerical Values and Ranges
- Exact match between generated value and reference specific value: rate as 10.
- Generated range/value falls within reference range: rate as 10.
- Generated value differs from reference specific value: deduct points based on relative deviation (e.g., minor difference within 10-20% → 8-9; moderate difference → 5-7; order-of-magnitude difference → 1-3).
- Reference gives a sequence and generated gives a sequence: compare overlap, trend direction, and magnitude similarity. High overlap with matching trend → 8-10; partial overlap → 4-7; minimal overlap → 1-3.
- **Important**: If the reference describes only a qualitative trend (e.g., "increases with radius") and the generated output provides specific numerical values, you cannot award a 10. The generated content, while possibly more detailed, does not "match" the reference in type or intent. Rate based on whether the numerical sequence supports the described trend. If it does, rate 7-8; if it contradicts, rate lower.

### Descriptive Content
- Perfect match in meaning and all key details: rate as 10.
- Minor differences in wording or degree (e.g., "sharp" vs. "very sharp") but core meaning is identical: rate 8-9.
- Partial match with some missing or slightly incorrect details: rate 4-7.
- Major discrepancies or contradictions: rate 1-3.
- Completely unrelated content: rate as 0.

### Empty or Missing Fields
- If EITHER generated or reference field is empty, null, or missing: rate as 0.

### Consistency Principles
- Focus on semantic meaning rather than exact wording, but do not ignore factual discrepancies.
- Providing extra information beyond the reference does NOT justify a higher score. Rate only on alignment with what the reference contains.
- Apply the same standards consistently across all categories.
